# Continue configs

Auto-generated, curated model configs for the [Continue](https://docs.continue.dev) VS Code extension.
Provides **fast**, **review**, and **thinking** tiers of remote LLMs (Groq, SambaNova, OpenRouter) with ≥ 6 k tokens of context & completion budget.

## Features
- Fetches live model catalogues from Groq, SambaNova and OpenRouter
- Filters for models that support ≥ 6 k tokens in both context and completion
- Builds a ready-to-use `config.yaml` sorted into three sections:
  1. **Fast coding chatter** – quick edits, small snippets
  2. **Code review chatter** – docs, refactors, multi-file diffs
  3. **Deep thinking** – massive context, big-brain analysis
